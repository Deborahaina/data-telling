Household Income

She uses the javaScript leaflet mapping library in conjunction with mapbox tile layer to create a choropleth map which visualizes the median household incomes cities in New York, New Jersey and Connecticut. A choropleth map is one in which areas are shaded or patterned in proportion to the statistical variable being represented. The choropleth map provides an easy way to visualize how a measurement varies across a geographic area, showing the level of variability within a region. This api call is rather slow in its data retrieval because of the size of data it returns.

The data is sourced from the beta.nyc open data site.
